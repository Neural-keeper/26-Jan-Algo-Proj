# Month 1 — Algorithm Arena (Algorithms & Data Structures)

### Week 1 — Setup & Baselines

- Define 5–7 core problems (sorting, shortest path, range queries)
    
- Implement naïve baseline solutions
    
- Build input generators (random, adversarial, real-world-like)
    
- CLI runner skeleton
    

### Week 2 — Competitive Implementations

- Implement multiple algorithms per problem
    
- Add timing, memory tracking, and correctness checks
    
- Store results in structured logs
    

### Week 3 — Visualization & “Fun”

- Add head-to-head comparisons
    
- Build heatmaps, race charts, leaderboards
    
- Introduce scoring (speed, memory, stability)
    

### Week 4 — Usefulness

- Add “recommendation mode” for new workloads
    
- Write short analysis: _why algorithm X wins here_
    
- Final demo: algorithm tournament replay
